# football-match-comic

# Football Match Comic 足球赛事漫画生成工具

## 项目简介

**football-match-comic** 是一套专业的**足球赛事转日式纪实漫画**自动化创作能力，依托标准化流程、多渠道事实核验、全媒体素材挖掘与日式运动漫画创作规范，可将足球赛事战报、实时赛况、赛事笔记、锦标赛事件等内容，批量输出**事实严谨的黑白日式漫画脚本、分镜规划、故事大纲、图像生成提示词**。
本能力支持单场赛事回顾、世界杯/联赛系列纪实漫画、德比大战、点球大战、球队出征/出局、球迷向社交滑动漫画等多类场景，兼顾**事实准确性**与**漫画叙事表现力**，适用于赛事二创、体育自媒体、球迷内容、赛事纪实连载等使用场景。

## 核心能力

1. **多场景适配**
   - 单场赛事：完成赛事卡片、时间线、关键节点、漫画脚本与配图提示词
   - 赛事系列专题：将世界杯、欧冠等锦标赛事件制作成长篇连载纪实漫画单集
   - 社交内容：适配短视频/图文平台竖版滑动漫画、四格趣味漫画、赛事封面海报
2. **双重事实保障**
   采用**官方权威信源+全网素材**双调研机制，严格区分客观赛事事实、公众舆论、艺术演绎，杜绝虚构赛果、进球、判罚等核心信息。
3. **全媒体素材挖掘**
   定向检索**微博、抖音、小红书、B站**四大中文主流平台+综合体育媒体，抓取球迷反响、赛场名场面、热梗、二创素材、现场氛围等创作灵感。
4. **标准化日式漫画创作**
   遵循经典黑白日式运动漫画风格，支持非制式分镜、镜头设计、情绪分镜、节奏把控，输出符合行业规范的漫画页规划、脚本、AI绘图提示词。
   
   ## 适用场景
- 体育自媒体：赛事复盘漫画、赛后趣味条漫、热门赛事短视频配图
- 球迷二创：世界杯/联赛纪实连载漫画、名场面复刻漫画
- 内容运营：社交平台滑动式赛事漫画、赛事宣传海报、观赛图文内容
- 赛事纪实：锦标赛系列纪录片漫画、球队出征/淘汰专题漫画
  
  ## 功能模式
  
  ### 1. 单场赛事回顾模式
  
  针对**已结束/进行中**单场足球比赛，输出完整轻量化内容包：
- 核验版赛事时间线、比分、关键事件
- 赛事核心亮点提炼
- 漫画页面规划、分镜脚本
- 配套AI图像生成提示词
  
  ### 2. 锦标赛系列专题模式
  
  针对世界杯、欧冠、洲际杯等大型赛事的**专题事件**（球队出征、小组赛爆冷、决赛点球大战、球星告别、主办城市筹备等），将事件作为连载漫画单集创作，支持长篇剧集化输出。
  
  ## 工作流程
  
  本工具默认采用**分阶段标准化流程**（可按需开启快速一键输出模式），全流程环环相扣，兼顾严谨性与创作性：
1. **模式判定**：区分单场赛事 / 赛事系列专题
2. **事实核验**：依托官方赛事页面、足联公告、权威战报核实赛事核心信息
3. **首轮素材检索**：同步完成四大中文平台素材挖掘，输出素材雷达表
4. **补充征集**：收集用户自定义关键词、截图、名场面、创作角度等补充素材
5. **素材整合筛选**：整合全网视觉、情绪、舆论素材，剔除无效内容
6. **故事主线搭建**：确立叙事主角、戏剧冲突、悬念、转折点与情感余韵
7. **大纲与页面规划**：根据剧情节奏确定漫画页数、逐页戏剧定位
8. **方案确认**：与用户核对页面规划、风格、视觉焦点，支持调整修改
9. **正式脚本创作**：编写标准漫画分镜脚本（镜头、动作、字幕、对白）
10. **导演级优化**：优化镜头、留白、反应分镜，强化漫画叙事感
11. **生成提示词**：按页面输出适配黑白日式漫画的AI绘图提示词
12. **边界标注**：区分客观事实与漫画艺术演绎

    ## 风格与版式规范
    
    ### 主流输出风格
16. **黑白日式足球漫画（默认）**
    白底墨线、网点阴影、速度线、非均匀分镜，写实球员肢体与赛场氛围，无花哨特效，适配纪实向内容。
17. **热血运动漫**：夸张动作、英雄特写、强情绪镜头，适配高光进球、绝杀等名场面。
18. **四格趣味漫画**：精简分镜、反差反应、趣味字幕，适合社交短内容。
19. **悲情/史诗风**：留白构图、克制镜头，适配球队出局、告别、决赛等重情绪场景。
    
    ### 页面版式
- 社交滑动漫画：默认 **3:4 竖版**，单页多格非制式分镜，拒绝均等网格布局
- 核心名场面：设置超大主视觉分镜，搭配特写、反应镜、空镜穿插
- 文字规则：字幕精简，保留球队、人物、时间、比分等核心标识，单页信息点不超过8个
  
  ## 素材检索规则
  
  ### 必检索中文平台（强制要求）
  
  所有赛事创作均需扫描四大平台，区分**事实内容**与**创作素材**：
1. 微博：热搜、话题、媒体/球迷评论、热梗
2. 抖音：短视频名场面、剪辑风格、热门片段
3. 小红书：观赛打卡、球迷日常、穿搭、场馆内容
4. B站：赛事复盘、解说、二创合集、弹幕热点
   
   ### 检索原则
- 官方信源 = 唯一事实依据，社交平台内容仅作为**创作灵感**
- 存疑信息标注「待核验/艺术演绎」，严禁虚构赛事数据、判罚、球员行为
  
  ## 事实安全约束
1. 禁止虚构进球球员、进球时间、红黄牌、VAR判罚、点球主罚者等核心赛事信息。
2. 赛事进行中需标注信息时间戳，不使用“最终获胜”等定论式描述。
3. 非商用创作默认去除赞助商Logo、水印、官方赛事商标，仅保留球队球衣、配色、人物特征。
4. 转播画面、用户提供的细节内容，统一标注为「视觉演绎」，不认定为客观事实。
   
   ## 快速使用指引
5. 确定创作类型：单场赛事复盘 / 锦标赛专题连载
6. 提供基础信息：赛事名称、对阵双方、比赛时间、核心事件（可选补充关键词/素材）
7. 等待首轮事实包+平台素材结果，按需补充创作角度
8. 确认漫画页数、叙事主线、视觉风格
9. 接收最终内容：漫画脚本 + 分镜规划 + AI绘图提示词
   
   ## 拓展说明
- 支持**快速一键输出**模式：跳过分步确认环节，直接产出完整内容
- 支持自定义尺寸：可适配 9:16、16:9 等多比例页面（短视频/公众号头图）
- 支持封面创作：可为章节开篇、决赛、特别篇制作专属漫画海报封面
- 全内容默认中文输出，可按需切换外文脚本与字幕
